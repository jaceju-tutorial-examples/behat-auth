# Test Authentication by PHPUnit and Behat

## Installation

```bash
git clone https://github.com/jaceju/example-auth-phpunit-behat.git
cd example-auth-phpunit-behat
php -r "copy('.env.behat', '.env');"
composer install
```

## PHPUnit

```bash
./vendor/bin/phpunit
```

## Behat

```bash
./vendor/bin/behat
```
